# Rendey LLC — Website (Next.js + Tailwind)

This is a premium, Stripe-friendly website for **Rendey LLC** (U.S.-registered company).

## Quickstart

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push to GitHub.
2. Import the repo in Vercel.
3. Deploy.

## Contact form

The contact form uses **Formspree** by default.
- Open `app/contact/page.tsx`
- Replace the `action` URL with your Formspree form endpoint.

You can also swap it for Resend (Vercel) later.

## Company info

Business address and support phone are in `components/site-config.ts`.
